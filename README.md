This repository provides a lightweight demo of our robotics data visualization dashboard.

## Tech Stack

Core tools used in this project:

*   **[Lit](https://lit.dev/)**: A simple and fast library for building lightweight, standard-compliant web components.
*   **[Shoelace](https://shoelace.style/)**: A forward-thinking library of professionally designed UI components, providing a consistent and polished look.
*   **[Apache ECharts](https://echarts.apache.org/)**: A powerful, interactive charting and data visualization library.
*   **[Vite](https://vitejs.dev/)**: The frontend build tool used to ensure lightning-fast development with Hot Module Replacement (HMR).


## Getting Started

Follow these steps to set up the project on your local machine.

### Prerequisites

This project is built with **Vite**. Ensure you have [Node.js](https://nodejs.org/en/download) installed (**version 20.19+, 22.12+**).

### Installation

#### 1. Clone the repository

```bash
git clone https://github.com/your-repo/data-viz.git
cd data-viz
```

#### 2. Install dependencies

Install all required packages listed in `package.json`:

```bash
npm install
```

### Development

Launch the local development server (typically available at http://localhost:5173):

```bash
npm run dev
```

#### Build & Deployment

```bash
# Build the project for production
npm run build

# Preview the production build locally
npm run preview
```
